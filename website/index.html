What is AlphaZero Light?

AlphaZero Light is a clean, educational implementation of the AlphaZero algorithm. Unlike complex, highly optimized production systems, this project focuses on readability and understanding while still achieving superhuman performance in games like Gomoku (Five in a Row).

It combines Monte Carlo Tree Search (MCTS) for lookahead planning with a Deep Residual Neural Network for position evaluation and move selection.
